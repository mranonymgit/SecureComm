# SecureComm

Este es un nuevo proyecto que se dedica a ofrecer servicios de seguridad, foros de casos en la comunidad y problemas que necesitan solución, esto usando Flutter para ofrecer buen rendimiento en ambos sistemas operativos siendo eficiente en la utilidad de la aplicación.

This is a new project dedicated to providing security services, community forums for cases and problems that need solving, using Flutter to offer good performance on both operating systems while being efficient in the utility of the application.
